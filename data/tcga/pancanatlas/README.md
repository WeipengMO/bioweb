# TCGA PanCanAtlas Data

Data source: NCI GDC TCGA PanCanAtlas publication page.

Official page: https://gdc.cancer.gov/about-data/publications/pancanatlas

## Directory Layout

- `manifests/`: GDC manifest files and downloaded page snapshots.
- `raw/`: original downloaded files from GDC, unchanged.
- `processed/`: converted Parquet/DuckDB-ready tables generated by local processing scripts.
- `metadata/`: local registry, checksums, and download logs.

## Recommended App Data

For the current BioWeb features, the minimum useful PanCanAtlas files are:

- RNA expression matrix:
  - `EBPlusPlusAdjustPANCAN_IlluminaHiSeq_RNASeqV2.geneExp.tsv`
  - Used for gene expression signature, correlation, tumor/normal comparison.
- Clinical outcome:
  - `TCGA-CDR-SupplementalTableS1.xlsx`
  - Recommended by GDC for survival endpoints.
- Clinical follow-up:
  - `clinical_PANCAN_patient_with_followup.tsv`
  - Useful as secondary clinical annotations.
- Merged sample quality annotations:
  - `merged_sample_quality_annotations.tsv`
  - Useful for excluding problematic samples.

## Download Commands

From the repository root:

```bash
source .venv/bin/activate
python scripts/data/download_pancanatlas.py metadata
python scripts/data/download_pancanatlas.py core
```

If the network is slow, use the local proxy:

```bash
export https_proxy=http://172.18.50.109:7890
export http_proxy=http://172.18.50.109:7890
```

`metadata` downloads the GDC publication page snapshot, open-access manifest, and controlled-access manifest link target when accessible.

`core` downloads small core files needed for survival metadata and sample filtering. It intentionally skips the large RNA expression matrix unless requested.

To download the large RNA expression matrix:

```bash
source .venv/bin/activate
python scripts/data/download_pancanatlas.py dataset rna_expression
```

This has already been downloaded in the current workspace.

To download every registered open supplemental file:

```bash
source .venv/bin/activate
python scripts/data/download_pancanatlas.py all-open
```

## Controlled Access

The controlled-access manifest on the GDC page requires dbGaP/GDC authorization and a token. This project records the link but does not automate controlled download by default.

## Sources Used

The GDC page describes PanCanAtlas as comparing 33 TCGA tumor types and lists open supplemental files including RNA, RPPA, methylation, miRNA, copy number, mutation MAF, TCGA-CDR clinical outcome, clinical follow-up, sample quality annotations, and PARADIGM matrix.
